684  * EntryMode[7:0] = ORG - 0 - ID1 - ID0 - AM - 0 - 0 - 0
685  ORG = 0, the origin address is not moved
686  ORG = 1, the original address 00000h moves according to the I/D[1:0] setting
687  AM = 1, the address is updated in vertical [landscape] writing direction
688  AM = 0, the address is updated in horizontal [portrait] writing direction
689  ID[1:0] = address counter (AC) to automatically increase or decrease by 1 when update one pixel
690  00 H- / V-

693  11 H+ / V+ : portrait = left to right
694  * Driver Output Control[10:8] = SM - 0 - SS

696  When SS = 1, the shift direction of outputs is from S720 to S1.
