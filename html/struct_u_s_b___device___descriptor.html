Detailed Description

USB Device Descriptor.
The device descriptor of a USB device represents the entire device. As a result a USB device can only hav e one device descriptor. It specifies some basic, yet important information about the device such as the s upported USB version, maximum packet size, vendor and product IDs and the number of possible configuration s the device can have.
