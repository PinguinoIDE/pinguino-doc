44  +5V A0 -|o |- VDD +5V

53  SYMBOL PIN DESCRIPTION NB
54  A0 1 address input 0 adress = 0 1 0 0 A2 A1 A0 0
55  A1 2 address input 1 A0, A1 et A2 relies au +5V
56  A2 3 address input 2 donc adress = 01001110 = 0x4E
57  P0 4 quasi-bidirectional I/O 0 LCD_BL
