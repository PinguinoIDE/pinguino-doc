This togehter with the coresponding usbconfig.\-c file needs to be filled up by the user according to implem ent the U\-S\-B functionality. The names are fixed since they are used inside the usb.\-h and usb.\-c file.\par
 E\-P0 is always filled up by the library so you dont really need to worry about. The user just needs to prov ide the right device, config and interface descriptor for it.\par
 The following things need to be filled up by the user (see pic\-U\-S\-B.\-h for a description of the structs)
\begin{DoxyItemize}
\item The device descriptor.
\item One configuration descriptor (multiple configuration not yet suppoerted)
\item One interface descriptor for each desired interface.
\item One endpoint descriptor for each endpoint in each interface.
