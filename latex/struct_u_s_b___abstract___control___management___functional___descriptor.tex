\subsection{Detailed Description}
Abstract Control Management Functional Descriptor The Abstract Control Management functional descriptor describes the commands supported by the Communication Class interface, as defined in Section 3.\-6.\-2, with the Sub\-Class code of Abstract Control Model. It can only occur within the class-\/specific portion of an Interface descriptor.
