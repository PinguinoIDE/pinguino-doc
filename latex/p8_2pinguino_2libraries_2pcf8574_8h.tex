\subsubsection*{-\/-\/-\/-\/------ Ecriture d'un octet sur le port du P\-C\-F8574 via le bus I2\-C }In Master Transmitter mode, the first byte transmitted contains the slave address of the receiving device (seven bits) and the Read/\-Write (R/\-W) bit. In this case, the R/\-W bit will be logic ‘0’. Serial data is transmitted eight bits at a time. After each byte is transmitted, an Acknowledge bit is received. Start and Stop conditions are output to indicate the beginning and the end of a serial transfer.
